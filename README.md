# Trip Organizer

Group event planner for outdoor trips with weather contingency planning.

## What It Does

- Set primary + backup weekends for events 6 months out
- SMS-based authentication and notifications
- Live RSVP tracking with availability heatmaps
- Single source of truth for logistics, gear, meals, parking
- Minimal SMS notifications (invite, final decision, critical updates)
- Switch between primary/backup based on weather/turnout

## Features

- **Availability Management**: Mark which weekend(s) you can attend
- **Event Hub**: All trip details, logistics, and resources in one place
- **SMS Notifications**: Phone verification + critical updates only
- **Weather Contingency**: Easy switch to backup weekend
- **Gear & Meal Coordination**: Sign-up tracking for shared resources

## Tech Stack (Proposed)

- **Frontend**: Next.js + React + Tailwind CSS
- **Backend**: Supabase (Auth + Database)
- **SMS**: Twilio
- **Hosting**: Vercel
